# Project Overview

This web app is built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Getting Started

1) Install dependencies
```sh
npm install
```

2) Run the dev server
```sh
npm run dev
```

3) Build for production
```sh
npm run build
```

## Tech Stack
- Vite + React + TypeScript
- Tailwind CSS + shadcn-ui
- React Router
- TanStack Query

## Notes
- Environment variables live in `.env` / `.env.example`.
- Static assets are served from `public/`.
- App entry points are in `src/main.tsx` and `src/App.tsx`.
